# Odin Recipes

A basic recipe website built with HTML as part of The Odin Project.

## About

This project is my first HTML project. The goal is to practice the fundamentals of HTML by building a simple multi-page recipe website.

## Features

* Home page with links to recipes
* Multiple recipe pages
* Images for each recipe
* Ingredients and cooking steps
* Navigation between pages

## Skills Practiced

* HTML boilerplate
* HTML elements
* Headings and paragraphs
* Lists
* Links
* Images
* File and folder organization
* Git and GitHub

## Future Improvements

As I continue learning, I plan to improve this project by adding:

* CSS styling
* Responsive design
* Better page layout
* Improved navigation
* More recipes
